# Limitations

This first version may not accurately model every Arena resolution, interface state, alternate card style, foil animation, mobile screenshot, paper card, MTGO screenshot, complex land, modal card, treasure, mana creature, cost reduction, alternate casting cost, or sideboarded configuration unless the submitted main deck is edited first.

Screenshot recognition is conservative and must be manually confirmed. Synthetic screenshots do not prove real Arena accuracy.

The app does not provide professional keep-or-mulligan advice. It displays transparent statistics and simple rule-based interpretation only.
